# CLAUDE.md - EEZO消費者行動×収益シミュレーション

## プロジェクト概要
- **目的**: 知覚品質向上施策がWTP・CVR・LTVに与える影響をモデル化し、売上10百万円達成への道筋を可視化
- **アプローチ**: 消費者行動モデル（心理→行動）と収益構造モデル（購買→LTV）の多層シミュレーション
- **評価指標**: CVR改善率、WTP変化、LTV、累積粗利額

---

## Core Philosophy

### 1. 因果モデル優先
単純な「販売数×単価」ではなく、「なぜ売れるか」の因果関係をモデル化する。

### 2. エビデンスベース
パラメータは可能な限りメタ分析の効果量（相関係数r）を使用する。

### 3. 保守的推計
- 購買意図→実購買の転換率: r=0.51（約半分）
- WTP表明→顕示の割引: 43-55%

---

## 多層モデル構造

```
┌─────────────────────────────────────────────────────────┐
│ Layer 1: 知覚品質モデル                                  │
│                                                         │
│  施策投入 → 知覚品質変化 → 購買意図変化                 │
│  (写真/レビュー/同梱カード等)                            │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│ Layer 2: 行動転換モデル                                  │
│                                                         │
│  購買意図 → 実購買（CVR）                               │
│  ※意図→行動の転換率 r=0.51 を適用                       │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│ Layer 3: 収益構造モデル                                  │
│                                                         │
│  初回購買 → リピート → LTV → 累積売上                  │
│  ※リピート率は同梱カード・開封体験の関数                 │
└─────────────────────────────────────────────────────────┘
```

## シミュレーション項目

### Sim1: ベースライン vs 改善後の比較
**目的**: 現状（商品置いてあるだけ）vs 施策投入後の差分を可視化

**入力**:
- 現状CVR: 0.04%
- 年間トラフィック（メルマガ）: 400,000クリック
- 年間トラフィック（O2O）: 50,000人

**計算**:
```python
# 現状
sales_baseline = traffic * cvr_baseline * avg_price
profit_baseline = sales_baseline * margin_rate

# 改善後
cvr_improved = cvr_baseline * (1 + photo_effect) * (1 + review_effect) * (1 + ux_effect)
sales_improved = traffic * cvr_improved * avg_price
profit_improved = sales_improved * margin_rate
```

**出力**: 
- `01_baseline_vs_improved.png` - 売上・粗利の比較棒グラフ
- CVR改善の内訳ウォーターフォール

---

### Sim2: 障壁クリアのファネル可視化
**目的**: 認知→興味→検討→購買の各段階で何%が脱落し、施策でどう変わるか

**ファネル段階**:
1. 認知（サイト訪問）: 100%
2. 興味（商品ページ閲覧）: 現状30%→改善後45%
3. 検討（カート追加）: 現状15%→改善後25%
4. 購買（決済完了）: 現状0.04%→改善後X%

**施策マッピング**:
| 段階 | 障壁 | 施策 | 期待改善率 |
|------|------|------|-----------|
| 認知→興味 | 「何これ？」 | 商品写真強化 | +50% |
| 興味→検討 | 「信頼できる？」 | レビュー・認証 | +67% |
| 検討→購買 | 「高い/面倒」 | UX改善・送料明示 | +25% |

**出力**:
- `02_funnel_comparison.png` - ファネル図（Before/After）
- 各段階の改善インパクト表

### Sim3: WTP感度分析
**目的**: 知覚品質向上によるWTP（支払意思額）変化のシミュレーション

**モデル**:
```python
# 効果量rから購買意図への影響を推計
purchase_intent_change = base_intent * (1 + effect_size_r)

# 意図→行動の転換
actual_purchase_prob = purchase_intent_change * 0.51

# WTP変化（表明→顕示の割引適用）
wtp_revealed = wtp_stated * 0.55  # 45%割引
```

**シナリオ**:
| シナリオ | 施策 | WTP変化（表明） | WTP変化（顕示推計） |
|----------|------|-----------------|-------------------|
| A | 写真のみ | +15% | +8% |
| B | 写真+レビュー | +30% | +16% |
| C | 写真+レビュー+同梱カード | +45% | +25% |

**出力**:
- `03_wtp_sensitivity.png` - シナリオ別WTP変化
- 投資対効果のROI曲線

### Sim4: LTV・リピートモデル
**目的**: 同梱カードがリピート率・LTVに与える影響をモデル化

**LTV計算式**:
```python
ltv = avg_order_value * (1 + repeat_rate / (1 - repeat_rate)) * margin_rate
```

**シナリオ**:
| シナリオ | リピート率 | 平均購入回数 | LTV |
|----------|-----------|-------------|-----|
| 現状 | 10% | 1.11回 | 833円 |
| 同梱カードあり | 20% | 1.25回 | 938円 |
| 同梱カード+開封体験 | 30% | 1.43回 | 1,071円 |

**出力**:
- `04_ltv_scenarios.png` - シナリオ別LTV比較
- 累積顧客価値の時系列グラフ

### Sim5: 10百万円達成シナリオ
**目的**: 売上目標達成に必要な条件の逆算

**目標**: 年間売上10,000,000円、粗利1,500,000円

**逆算**:
```python
required_customers = target_sales / (avg_price * avg_purchases_per_customer)
required_traffic = required_customers / cvr_target
required_cvr = required_customers / available_traffic
```

**制約条件**:
- 利用可能トラフィック: メルマガ400,000 + O2O 50,000 + SNS 100,000 = 550,000
- 粗利率: 15%

**出力**:
- `05_target_achievement.png` - 達成シナリオマトリクス
- 必要CVR×必要単価×必要リピート率の3軸グラフ

## 補足：心理学的根拠

### 情報ギャップ理論（Loewenstein 1994）
好奇心は「知っていることと知りたいことのギャップ」から生じる。同梱カードの「話のネタ」設計はこれを活用。

### 開封体験の両刃の剣（Joutsela 2016）
- 57%でWTP上昇
- 14%で低下
- **やるなら徹底的に、しないなら割り切る**

### 信頼の普遍性（Handoyo 2024）
信頼の効果は国の所得水準や消費者セグメントに関係なく普遍的。信頼構築への投資は最もロバスト。
